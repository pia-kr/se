# Mini Drive (클라우드 파일 공유 시스템)

## 프로젝트 소개
조직 내 파일을 효율적으로 저장하고 공유하기 위한 클라우드 기반 파일 관리 시스템입니다.

---

## 주요 기능
- 파일 업로드 및 다운로드
- 폴더 생성, 수정, 삭제 및 계층 구조 관리
- 사용자 간 파일 공유 및 권한 설정
- 링크 기반 외부 파일 공유 (만료 기간 설정)
- 파일 검색 기능 (이름, 날짜, 사용자, 유형 기준)
- 파일 버전 관리 (이전 버전 조회 및 복원)
